## 3.3.1 Word 与 WPS 文字

![AI辅助](../../../图片/ai.svg)

Word 和 WPS 文字属于“文字处理软件”，主要用于写文章、写通知、写总结、写简历、排版作业。二者界面略有不同，但最常用的功能和操作思路非常接近。

>[!TIP]
>如果你使用的是 WPS 文字，也可以直接参考本页内容，因为按钮位置可能略有区别，但名称通常相近。

### 3.3.1.1 新建与保存文档

![简单](../../../图片/easy.svg)

打开软件后，通常会看到“空白文档”按钮，单击即可开始写内容。第一次新建文档后，建议立刻保存一次。

- 点击“文件”
- 选择“保存”或“另存为”
- 选择保存位置
- 输入清楚的文件名

建议文件名不要只写“新建文档”“作业”“通知”，更推荐：
- `高一语文作文-张三`
- `社团招新通知`
- `求职简历-李四`

#### 保存格式说明

Word 默认 `.docx`，WPS 默认 `.wps`，都支持导出 `.docx`。

| 格式 | 说明 |
| --- | --- |
| `.docx` | Office 2007+ 默认格式，支持更多新功能，体积较小 |
| `.doc` | Office 2003 及以前格式，兼容性好但功能较少 |
| `.wps` | WPS 默认格式，在 WPS 中体验最佳 |
| `.txt` | 纯文本，无格式，适合简单记录 |
| `.pdf` | 固定格式，适合最终发布，不适合继续编辑 |

保存时注意：
- 重要文档建议同时保存 `.docx`（便于继续编辑）和 `.pdf`（便于分享）
- 文档中包含特殊字体时，最好嵌入字体或导出 PDF
- 定期保存（按 `Ctrl + S`）是避免意外丢失的最佳习惯

#### 自动保存功能

现代版本的 Word 和 WPS 都带自动保存功能，通常每几分钟自动保存一次。但自动保存不能替代手动保存，建议：
- 重要文档每完成一个段落就保存一次
- 设置较短的自动保存间隔（如 5 分钟）
- 了解自动保存文件的存放位置，以便崩溃后恢复

### 3.3.1.2 输入与修改文字

![简单](../../../图片/easy.svg)

把光标放到页面中，直接打字即可。修改已有内容时：
- 用鼠标拖动选中文字
- 删除、替换、复制、粘贴

常用快捷键：

| 功能 | 快捷键 |
| --- | --- |
| 复制 | `Ctrl + C` |
| 粘贴 | `Ctrl + V` |
| 剪切 | `Ctrl + X` |
| 撤销 | `Ctrl + Z` |
| 保存 | `Ctrl + S` |

这些快捷键几乎所有办公软件都能用到。

#### 光标移动

| 操作 | 快捷键 |
| --- | --- |
| 行首 / 行尾 | `Home` / `End` |
| 文档开头 / 结尾 | `Ctrl + Home` / `Ctrl + End` |
| 向上 / 向下翻页 | `Page Up` / `Page Down` |
| 向左 / 向右移动一个字符 | `←` / `→` |
| 向左 / 向右移动一个单词 | `Ctrl + ←` / `Ctrl + →` |

#### 选中文字

- 双击鼠标：选中一个单词
- 三击鼠标：选中整个段落
- `Ctrl + 点击`：选中整句
- `Shift + 点击`：选中光标起始位置到点击位置
- `Ctrl + A`：选中整个文档

#### 撤销与恢复

- `Ctrl + Z`：撤销上一步操作
- `Ctrl + Y`：恢复被撤销的操作
- 可以撤销多步操作，但有一定限制

### 3.3.1.3 排版与格式

![简单](../../../图片/easy.svg)

不要靠空格对齐文字，也不要靠回车“顶”位置。这样的排版在换电脑后容易乱。优先使用“对齐”“段落”“表格”等正规工具。

常见排版项目：

| 项目 | 用途 |
| --- | --- |
| 字号 | 标题用大号，正文用适中字号 |
| 字体 | 建议全篇统一，避免频繁切换 |
| 加粗 | 突出重点内容 |
| 倾斜 | 强调词语或短语 |
| 下划线 | 标记特殊内容，不宜过多 |
| 对齐方式 | 左对齐、居中、右对齐、两端对齐 |
| 行距 | 推荐 `1.25~1.5` 倍 |
| 段前段后间距 | 提高段落可读性 |
| 项目符号 / 编号 | 列举时更清晰 |
| 分栏 | 适合简历、通知等多栏内容 |
| 页面边距 | 一般默认即可，特殊排版可调整 |

常见排版建议：
- 标题用大号字号，居中并加粗
- 正文左对齐，段落首行可空两格
- 长文档分节、分标题，便于阅读
- 重要内容可用颜色、加粗突出，但不宜过多

> [!IMPORTANT]
> 不要靠反复敲空格来对齐文字，也不要靠反复敲回车来“顶”位置。建议使用“对齐”“段落”“表格”等正规工具。

#### 字号与字体

| 内容 | 建议 |
| --- | --- |
| 文档标题 | 二号或三号（22pt-24pt） |
| 段落标题 | 小四号或四号（14pt-16pt） |
| 正文 | 五号或小五号（10.5pt-12pt） |
| 辅助说明 | 六号（7.5pt） |

中文常用字体：
- 宋体：正式，适合正文
- 黑体：粗壮，适合标题
- 楷体：柔和，适合引用
- 仿宋：正式美观，适合公文
- 微软雅黑：屏幕阅读效果好

英文字体：
- Times New Roman：经典衬线字体
- Arial：无衬线，现代感强
- Calibri：Office 默认字体
- Consolas：等宽，适合代码

#### 对齐方式与段落设置

| 对齐方式 | 适用场景 |
| --- | --- |
| 左对齐 | 正文标准选择 |
| 居中 | 标题、封面、诗歌 |
| 右对齐 | 日期、签名、特殊排版 |
| 两端对齐 | 正式文档左右整齐 |
| 分散对齐 | 文字间距自动调整 |

段落设置可在“段落”对话框中调整：
- 首行缩进：中文文档通常空 2 个字符
- 悬挂缩进：编号列表常用
- 段前 / 段后间距：控制段落间隔
- 行距：单倍、1.5 倍、双倍或固定值

### 3.3.1.4 段落、列表与标题

![简单](../../../图片/easy.svg)

文档稍长时，建议分清段落和层次结构：
- 每讲一个内容另起一段
- 列表用项目符号或编号
- 长文档设置标题层级，便于阅读

写通知常见结构：
1. 标题
2. 正文说明
3. 时间地点
4. 联系方式
5. 落款

#### 标题层级

| 层级 | 说明 |
| --- | --- |
| 一级标题 | 文档大章节，如“第一章”“第一节” |
| 二级标题 | 大章节内小节，如“1.1”“1.2” |
| 三级标题 | 进一步细分，如“1.1.1” |

设置方法：
1. 选中标题文字
2. 在“样式”中选择“标题1”“标题2”等
3. 可自定义样式格式

#### 列表使用

| 列表类型 | 适用 |
| --- | --- |
| 项目符号 | 无顺序列举 |
| 编号列表 | 有顺序步骤或条目 |

项目符号：
- 符号样式可改为圆形、方形、箭头
- 可以使用图片作为符号

编号列表：
- 自动递增编号
- 支持多级编号

建议：
- 同一级别内容使用一致列表样式
- 列表项以短句为主
- 保持格式一致

### 3.3.1.5 插入图片与表格

![简单](../../../图片/easy.svg)

调整图片大小时，尽量拖动角上的控制点，而不是中间的边，这样不容易把图片变形。插入图片和表格可用于图文并排、课程表、登记表、说明文档。

#### 插入图片与表格

| 插入图片 | 插入表格 |
| --- | --- |
| 光标放到位置 | 点击“插入”菜单 |
| 选择“图片” | 选择“表格”，设置行列 |
| 找到图片并确认 | 在单元格中输入内容 |
| 调整大小和位置 | 可调整宽度、边框、颜色 |

常见用途：
- 课程表、值班表、登记表、对比表格
- 图文混排说明文档

> [!TIP]
> 插入表格后可用“合并单元格”“拆分单元格”实现复杂排版。

#### 图片格式与处理

| 格式 | 用途 |
| --- | --- |
| JPG | 照片类，体积小，有损压缩 |
| PNG | 图标、截图，支持透明背景 |
| GIF | 动图，适合简单动画 |
| BMP | 无压缩，质量高但文件大 |

处理功能：
- 裁剪：去除不需要部分
- 旋转：调整角度
- 压缩：减小文件体积
- 去除背景：删除背景（Office 2019+）

#### 文字环绕方式

| 方式 | 说明 |
| --- | --- |
| 嵌入型 | 图片作为字符插入，位置固定 |
| 四周型 | 文字环绕图片四周 |
| 紧密型 | 文字紧密环绕图片边缘 |
| 上下型 | 文字在图片上方或下方 |
| 穿越型 | 文字穿越图片空白区域 |

#### 表格基本操作

| 操作 | 说明 |
| --- | --- |
| 手动选择行列 | 插入表格 |
| 精确输入行列 | 插入表格 |
| Excel 粘贴 | 从 Excel 直接粘贴表格 |
| 合并单元格 | 选中多个单元格后右键合并 |
| 拆分单元格 | 将一个单元格分成多个 |
| 调整行高列宽 | 拖动边框或设置精确值 |
| 添加/删除行列 | 右键菜单操作 |

### 3.3.1.6 页面设置与打印

![简单](../../../图片/easy.svg)

如果文档要打印，通常需要检查页面设置。

检查内容：
- 纸张大小是否正确（如 A4、A3）
- 页面方向是纵向还是横向
- 页边距是否太小或太大
- 打印页数是否正确
- 是否需要页码、页眉页脚

打印前：
- 先用“打印预览”检查排版
- 确认表格、图片没有被截断
- 表头、页码是否完整

导出 PDF 可以保证格式稳定，适合发送给别人。

常见打印路径：`文件` -> `打印` 或 `导出` -> `导出为PDF`

#### 纸张与页边距

| 纸张 | 说明 |
| --- | --- |
| A4 | 最常用，适合大多数情况 |
| A3 | 双页打印或大幅面文档 |
| Letter | 美国标准信纸 |
| 16开 | 国内常见杂志尺寸 |
| 自定义 | 特殊需求时使用 |

标准页边距：
- 上：2.54 厘米
- 下：2.54 厘米
- 左：3.17 厘米
- 右：3.17 厘米

装订线：需要装订的文档在左侧增加额外边距

#### 页眉页脚与分节符

页眉页脚常用于：
- 添加页码
- 添加文档标题
- 添加公司标志
- 添加日期时间

设置方法：
1. 双击页面顶部或底部
2. 进入页眉/页脚编辑模式
3. 插入所需内容
4. 关闭页眉页脚

分节符可以设置不同页面格式：
- 不同页眉页脚
- 不同页面方向
- 不同列数
- 不同页边距

### 3.3.1.7 初学者常见问题

![简单](../../../图片/easy.svg)

| 问题 | 可能原因 | 解决建议 |
| --- | --- | --- |
| 排版总是乱 | 用空格硬推位置、用回车硬调、图片尺寸拉得随意 | 优先使用“对齐”“段落”“表格”等正规工具 |
| 发给别人格式不一样 | 软件不同、缺少字体、排版方式不稳定 | 使用常见字体、导出 PDF、提前沟通软件版本 |
| 文字显示为方框 | 对方电脑未安装字体 | 转换为曲线（嵌入）、导出 PDF、使用常见字体 |
| 表格跨页断开 | 未设置允许跨页断行 | 右键“表格属性”，勾选“允许跨页断行”或“重复标题行” |
| 行距无法调整 | 段落设置固定行距，或图形/表格影响 | 检查段落设置中的行距选项，清除格式后重新设置 |

### 3.3.1.8 进阶技巧

#### 查找与替换

- 打开查找框：`Ctrl + F`
- 输入要查找的内容
- 使用“查找下一个”定位

替换功能：
- 打开替换框：`Ctrl + H`
- 输入查找内容和替换内容
- 选择“替换”或“全部替换”

高级选项：
- 区分大小写
- 全字匹配
- 使用通配符
- 格式搜索

#### 样式与格式刷

样式作用：
- 快速统一格式
- 方便修改全局样式
- 生成目录的基础

格式刷使用：
- 单击格式刷：复制一次格式
- 双击格式刷：可多次使用
- 清除格式：选中后点击清除格式

#### 目录自动生成

创建目录：
1. 为标题应用样式（标题1、标题2 等）
2. 插入目录
3. 选择目录样式
4. 更新目录

更新目录：
- 右键目录选择“更新域”
- 选择“更新整个目录”

#### 批注与修订

批注：
- 选中内容，插入批注
- 可回复、删除批注
- 适合提出建议或疑问

修订：
- 开启修订模式
- 所有修改会被记录
- 可接受或拒绝修改

#### 分栏排版

使用场景：
- 书籍、杂志排版
- 简历制作
- 通知文书

设置方法：
1. 选中要分栏的内容
2. 点击“布局”->“分栏”
3. 选择栏数
4. 可设置分隔线

#### 首字下沉

设置方法：
1. 光标放在段落中
2. 点击“插入”->“首字下沉”
3. 选择下沉方式
4. 设置字体、下沉行数

#### 脚注与尾注

脚注：
- 页面底部注释
- 用于解释说明

尾注：
- 文档末尾注释
- 用于引用来源

插入方法：
1. 光标放在要添加注释的位置
2. 点击“引用”->“插入脚注”或“插入尾注”

#### 邮件合并

用途：
- 批量生成邀请函
- 制作信封标签
- 生成成绩单

基本步骤：
1. 准备数据源（Excel 或 Word 表格）
2. 插入合并域
3. 预览结果
4. 完成合并

#### 文档比较与合并

比较文档：
- 审阅 -> 比较
- 显示两份文档的差异

合并文档：
- 审阅 -> 合并
- 将多个文档合并为一个

#### 快捷键汇总

文字编辑：
- `Ctrl+B`：加粗
- `Ctrl+I`：斜体
- `Ctrl+U`：下划线
- `Ctrl+Shift+>`：增大字号
- `Ctrl+Shift+<`：减小字号

段落格式：
- `Ctrl+L`：左对齐
- `Ctrl+E`：居中
- `Ctrl+R`：右对齐
- `Ctrl+J`：两端对齐

文件操作：
- `Ctrl+N`：新建文档
- `Ctrl+O`：打开文档
- `Ctrl+W`：关闭文档
- `Ctrl+P`：打印
