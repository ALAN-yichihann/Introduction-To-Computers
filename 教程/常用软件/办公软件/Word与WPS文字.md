## 3.3.1 Word 与 WPS 文字

Word 和 WPS 文字属于“文字处理软件”，主要用于写文章、写通知、写总结、写简历、排版作业。二者界面略有不同，但最常用的功能和操作思路非常接近。

>[!TIP]
>如果你使用的是 WPS 文字，也可以直接参考本页内容，因为按钮位置可能略有区别，但名称通常相近。

### 3.3.1.1 新建和保存文档

![简单](../../../图片/easy.svg)

打开软件后，通常会看到“空白文档”按钮，单击即可开始写内容。

第一次新建文档后，建议立刻保存一次：

1. 点击“文件”。
2. 点击“保存”或“另存为”。
3. 选择保存位置。
4. 给文件起一个清楚的名字。

建议文件名不要只写“新建文档”“作业”“通知”这种太模糊的名称，更推荐：

- `高一语文作文-张三`
- `社团招新通知`
- `求职简历-李四`

### 3.3.1.2 输入和修改文字

![简单](../../../图片/easy.svg)

把光标放到页面中，直接打字即可。

如果想修改已有内容：

- 用鼠标拖动选中文字
- 再进行删除、替换、复制、粘贴

最常用的快捷键有：

- `Ctrl + C`：复制
- `Ctrl + V`：粘贴
- `Ctrl + X`：剪切
- `Ctrl + Z`：撤销
- `Ctrl + S`：保存

这些快捷键几乎所有办公软件都能用到。

### 3.3.1.3 最常见的排版操作

![简单](../../../图片/easy.svg)

写文档时，最常碰到的几个按钮如下：

- 字号：控制字的大小
- 字体：控制字的样式
- 加粗：让重点更醒目
- 倾斜：用于强调
- 下划线：用于标记
- 对齐方式：左对齐、居中、右对齐
- 行距：让文字更疏或更密

常见场景举例：

- 标题一般用较大的字号，并居中
- 正文一般左对齐
- 多段文字看起来很挤时，可以适当调大行距

>[!IMPORTANT]
>不要靠反复敲空格来对齐文字，也不要靠反复敲回车来“顶”位置。这样换一台电脑后，格式往往就乱了。

### 3.3.1.4 段落、列表和标题

![简单](../../../图片/easy.svg)

文档稍长一些时，建议把内容分成清楚的段落和标题。

你可以这样做：

- 每讲一个新的内容，就另起一段
- 需要列出多条信息时，用项目符号或编号
- 长文档可以设置一级标题、二级标题，便于阅读

例如写通知时，常见结构可以是：

1. 标题
2. 正文说明
3. 时间地点
4. 联系方式
5. 落款

### 3.3.1.5 插入图片和表格

![简单](../../../图片/easy.svg)

如果你要做图文并排的作业、说明文档、课程表、登记表，就需要插入图片或表格。

插入图片的常见做法：

1. 把光标放到想插入图片的位置。
2. 点击“插入”。
3. 选择“图片”。
4. 从电脑中找到图片并确认。

插入表格的常见做法：

1. 点击“插入”。
2. 选择“表格”。
3. 选定行数和列数。
4. 在单元格中输入内容。

>[!TIP]
>调整图片大小时，尽量拖动角上的控制点，而不是中间的边，这样不容易把图片拉变形。

### 3.3.1.6 页面设置和打印

![简单](../../../图片/easy.svg)

如果文档要打印，通常还需要看一下页面设置。

常见要检查的内容有：

- 纸张大小是否正确
- 页面方向是纵向还是横向
- 页边距是否太小或太大
- 打印页数是否正确

打印的路径通常是“文件” -> “打印”。

如果只是想把内容稳定地发给别人看，通常更推荐导出成 PDF 再发送。

### 3.3.1.7 初学者常见问题

![简单](../../../图片/easy.svg)

#### 3.3.1.7.1 文档排版总是乱

常见原因包括：

- 用空格硬推位置
- 用回车硬调页面
- 图片尺寸拉得太随意

建议优先使用对齐、段落、表格这些正规工具来排版。

#### 3.3.1.7.2 发给别人后格式不一样

可能原因包括：

- 对方用的是另一套办公软件
- 电脑里缺少某些字体
- 你的排版方式本来就不稳定

如果只是给别人阅读或打印，导出 PDF 往往最省心。
