## 3.3.1 Word 与 WPS 文字

![AI辅助](../../../图片/ai.svg)

Word 和 WPS 文字属于“文字处理软件”，主要用于写文章、写通知、写总结、写简历、排版作业。二者界面略有不同，但最常用的功能和操作思路非常接近。

>[!TIP]
>如果你使用的是 WPS 文字，也可以直接参考本页内容，因为按钮位置可能略有区别，但名称通常相近。

### 3.3.1.1 新建和保存文档

![简单](../../../图片/easy.svg)

打开软件后，通常会看到“空白文档”按钮，单击即可开始写内容。

第一次新建文档后，建议立刻保存一次：

1. 点击“文件”。
2. 点击“保存”或“另存为”。
3. 选择保存位置。
4. 给文件起一个清楚的名字。

建议文件名不要只写“新建文档”“作业”“通知”这种太模糊的名称，更推荐：

- `高一语文作文-张三`
- `社团招新通知`
- `求职简历-李四`

#### 3.3.1.1.1 文件保存格式说明

Word 文档的默认格式是 .docx，这种格式在 Office 2007 及以上版本中通用。如果你使用的是较老版本的 Word（2003 及以前），保存时可能会看到 .doc 格式。WPS 文字默认保存为 .wps 格式，但也支持导出为 .docx 格式。

**不同格式的区别：**
- .docx：Office 2007+ 默认格式，支持更多新功能，文件体积相对较小
- .doc：Office 2003 及以前格式，兼容性更好但功能较少
- .wps：WPS 文字默认格式，可在 WPS 中获得最佳体验
- .txt：纯文本格式，无任何格式，适合简单记录
- .pdf：固定格式，适合最终发布，不适合继续编辑

**保存时的注意事项：**
- 重要文档建议同时保存一份 .docx（方便继续编辑）和一份 .pdf（方便分享）
- 如果文档中包含特殊字体，最好嵌入字体或转换为 PDF，否则可能在其他电脑显示异常
- 定期保存（按 Ctrl+S）是避免意外丢失的最佳习惯

#### 3.3.1.1.2 自动保存功能

现代版本的 Word 和 WPS 都带有自动保存功能，通常每几分钟会自动保存一次。但自动保存不能替代手动保存，建议：

- 重要文档每完成一个段落就保存一次
- 设置较短的自动保存间隔（如 5 分钟）
- 了解自动保存文件的存放位置，以便在崩溃后恢复

### 3.3.1.2 输入和修改文字

![简单](../../../图片/easy.svg)

把光标放到页面中，直接打字即可。

如果想修改已有内容：

- 用鼠标拖动选中文字
- 再进行删除、替换、复制、粘贴

最常用的快捷键有：

- `Ctrl + C`：复制
- `Ctrl + V`：粘贴
- `Ctrl + X`：剪切
- `Ctrl + Z`：撤销
- `Ctrl + S`：保存

这些快捷键几乎所有办公软件都能用到。

#### 3.3.1.2.1 光标移动技巧

除了直接点击定位光标外，还可以通过快捷键快速移动光标：

- `Home` / `End`：移动到行首 / 行尾
- `Ctrl + Home` / `Ctrl + End`：移动到文档开头 / 结尾
- `Page Up` / `Page Down`：向上 / 向下翻页
- `←` / `→`：向左 / 向右移动一个字符
- `Ctrl + ←` / `Ctrl + →`：向左 / 向右移动一个单词

#### 3.3.1.2.2 选中文字的多种方法

- 双击鼠标：选中一个单词
- 三击鼠标：选中整个段落
- `Ctrl + 点击`：选中整句
- `Shift + 点击`：选中光标起始位置到点击位置的所有内容
- `Ctrl + A`：选中整个文档

#### 3.3.1.2.3 撤销与恢复

- `Ctrl + Z`：撤销上一步操作
- `Ctrl + Y`：恢复被撤销的操作
- 可以撤销多步操作，但有一定限制

### 3.3.1.3 最常见的排版操作

![简单](../../../图片/easy.svg)

>不要靠反复敲空格来对齐文字，也不要靠反复敲回车来“顶”位置。这样换一台电脑后，格式往往就乱了。

写文档时，常见的排版操作和建议如下：

- **字号**：控制字的大小，标题建议用较大字号，正文用适中字号。
- **字体**：选择合适的字体，建议全篇统一，避免频繁切换。
- **加粗**：用于突出重点内容，如标题、关键词。
- **倾斜**：用于强调某些词语或短语。
- **下划线**：用于标记特殊内容，但不宜过多使用。
- **对齐方式**：常用左对齐、居中、右对齐。标题一般居中，正文左对齐。
- **行距**：让文字更疏或更密，推荐1.25~1.5倍行距，便于阅读。
- **段前段后间距**：适当增加段落间距，让内容更清晰。
- **项目符号/编号**：用于列举事项，条理更清楚。
- **分栏**：适合排版简历、通知等多栏内容。
- **页面边距**：一般默认即可，特殊排版可适当调整。

常见排版建议：

- 标题一般用较大的字号，并居中，建议加粗。
- 正文一般左对齐，段落开头可空两格。
- 多段文字看起来很挤时，可以适当调大行距和段间距。
- 长文档建议分节、分标题，便于查找和阅读。
- 重要内容可用颜色、加粗等方式突出，但不宜过多。

> [!IMPORTANT]
> 不要靠反复敲空格来对齐文字，也不要靠反复敲回车来“顶”位置。这样换一台电脑后，格式往往就乱了。建议使用“对齐”“段落”“表格”等正规工具。

#### 3.3.1.3.1 字号大小选择指南

常见的字号选择建议：

- 文档标题：二号或三号（22pt-24pt）
- 段落标题：小四号或四号（14pt-16pt）
- 正文：五号或小五号（10.5pt-12pt）
- 辅助说明：六号（7.5pt）

#### 3.3.1.3.2 常用字体介绍

**中文字体：**
- 宋体：最正式，适合正文
- 黑体：粗壮有力，适合标题
- 楷体：柔和亲切，适合引用或特殊场合
- 仿宋：正式美观，适合公文
- 微软雅黑：无衬线字体，屏幕阅读效果好

**英文字体：**
- Times New Roman：经典衬线字体，适合学术论文
- Arial：无衬线字体，现代感强
- Calibri：Office 默认字体，清爽易读
- Consolas：等宽字体，适合代码

#### 3.3.1.3.3 对齐方式的正确使用

- **左对齐**：正文标准选择，左侧整齐，右侧可能不齐
- **居中**：标题、封面、诗歌等
- **右对齐**：日期、签名、特殊排版
- **两端对齐**：书籍、正式文档，左右两侧都整齐
- **分散对齐**：类似两端对齐，但会调整字间距

#### 3.3.1.3.4 段落设置详解

在 Word 中，可以通过“段落”对话框进行详细设置：

- 首行缩进：中文文档通常空2个字符
- 悬挂缩进：编号列表常用
- 段前间距：标题与正文之间的间距
- 段后间距：段落之间的距离
- 行距：单倍、1.5倍、双倍或固定值

### 3.3.1.4 段落、列表和标题

![简单](../../../图片/easy.svg)

文档稍长一些时，建议把内容分成清楚的段落和标题。

你可以这样做：

- 每讲一个新的内容，就另起一段
- 需要列出多条信息时，用项目符号或编号
- 长文档可以设置一级标题、二级标题，便于阅读

例如写通知时，常见结构可以是：

1. 标题
2. 正文说明
3. 时间地点
4. 联系方式
5. 落款

#### 3.3.1.4.1 多级标题的设置

长文档应该建立清晰的多级标题结构：

- 一级标题：文档大章节，如“第一章”“第一节”
- 二级标题：大章节内的小节，如“1.1”“1.2”
- 三级标题：更细分的内容，如“1.1.1”

设置方法：
1. 选中标题文字
2. 在“样式”中选择“标题1”“标题2”等
3. 可以自定义样式格式

#### 3.3.1.4.2 项目符号与编号的使用

**项目符号：**
- 用于无顺序的列举
- 可以更改符号样式（圆形、方形、箭头等）
- 可以使用图片作为符号

**编号列表：**
- 用于有顺序的步骤或条目
- 自动递增编号
- 可以设置多级编号

**使用建议：**
- 同一级别的内容使用同一种列表样式
- 列表项内容不宜过长，以短句为主
- 保持列表项格式一致

### 3.3.1.5 插入图片和表格

![简单](../../../图片/easy.svg)

>调整图片大小时，尽量拖动角上的控制点，而不是中间的边，这样不容易把图片拉变形。

如果你要做图文并排的作业、说明文档、课程表、登记表，就需要插入图片或表格。

**插入图片的常见做法：**
1. 把光标放到想插入图片的位置。
2. 点击“插入”菜单。
3. 选择“图片”，从电脑中找到图片并确认。
4. 插入后可拖动调整大小和位置。

**插入表格的常见做法：**
1. 点击“插入”菜单。
2. 选择“表格”，设置行数和列数。
3. 在单元格中输入内容。
4. 可调整表格宽度、边框、颜色等。

**常见用途：**
- 课程表、值班表、登记表、对比表格等。
- 图文混排说明文档。

> [!TIP]
> 调整图片大小时，尽量拖动角上的控制点，而不是中间的边，这样不容易把图片拉变形。
> 插入表格后可用“合并单元格”“拆分单元格”实现复杂排版。

#### 3.3.1.5.1 图片格式与处理

**常用图片格式：**
- JPG：照片类图片，体积小，有损压缩
- PNG：图标、截图，支持透明背景
- GIF：动图，适合简单动画
- BMP：无压缩，质量高但文件大

**图片处理功能：**
- 裁剪：去除不需要的部分
- 旋转：调整图片角度
- 压缩：减小文件体积
- 去除背景：删除图片背景（Office 2019+）

#### 3.3.1.5.2 文字环绕方式

图片插入后，需要设置文字环绕方式：

- 嵌入型：图片作为字符插入，位置固定
- 四周型：文字环绕图片四周
- 紧密型：文字紧密环绕图片边缘
- 上下型：文字在图片上方或下方
- 穿越型：文字穿越图片空白区域

#### 3.3.1.5.3 表格基本操作

**插入表格：**
- 手动拖动选择行列数
- 精确输入行列数
- 从Excel粘贴表格

**表格编辑：**
- 合并单元格：选中多个单元格后右键合并
- 拆分单元格：将一个单元格分成多个
- 调整行高列宽：拖动边框或设置精确值
- 添加删除行/列：右键菜单操作

### 3.3.1.6 页面设置和打印

![简单](../../../图片/easy.svg)


如果文档要打印，通常还需要看一下页面设置。

**常见要检查的内容有：**
- 纸张大小是否正确（如A4、A3等）
- 页面方向是纵向还是横向
- 页边距是否太小或太大
- 打印页数是否正确
- 是否需要页码、页眉页脚

**打印前建议：**
- 先用“打印预览”检查排版是否正常
- 确认表格、图片没有被截断
- 表头、页码等内容是否完整

**导出PDF：**
如果只是想把内容稳定地发给别人看，通常更推荐导出成 PDF 再发送，避免格式错乱。

**常见打印路径：**
“文件” -> “打印” 或 “导出” -> “导出为PDF”

#### 3.3.1.6.1 纸张大小选择

- A4：最常用，适合大多数情况
- A3：双页打印或大幅面文档
- Letter：美国标准信纸
- 16开：国内常见杂志尺寸
- 自定义：特殊需求时使用

#### 3.3.1.6.2 页边距设置

标准页边距：
- 上：2.54厘米
- 下：2.54厘米
- 左：3.17厘米
- 右：3.17厘米

装订线：需要装订的文档在左侧增加额外边距

#### 3.3.1.6.3 页眉页脚设置

页眉页脚常用于：
- 添加页码
- 添加文档标题
- 添加公司标志
- 添加日期时间

设置方法：
1. 双击页面顶部或底部
2. 进入页眉/页脚编辑模式
3. 插入所需内容
4. 关闭页眉页脚

#### 3.3.1.6.4 分节符的使用

分节符可以设置不同的页面格式：
- 不同的页眉页脚
- 不同的页面方向
- 不同的列数
- 不同的页边距

### 3.3.1.7 初学者常见问题

![简单](../../../图片/easy.svg)

#### 3.3.1.7.1 文档排版总是乱

常见原因包括：

- 用空格硬推位置
- 用回车硬调页面
- 图片尺寸拉得太随意

建议优先使用对齐、段落、表格这些正规工具来排版。

#### 3.3.1.7.2 发给别人后格式不一样


可能原因包括：
- 对方用的是另一套办公软件（如WPS与Word之间）
- 电脑里缺少某些字体，导致自动替换
- 你的排版方式本来就不稳定（如用空格、回车对齐）

**解决建议：**
- 尽量使用常见字体（如宋体、黑体、Arial等）
- 重要文档建议导出为PDF再发送，保证格式一致
- 复杂排版建议提前与对方沟通软件版本

#### 3.3.1.7.3 文字显示为方框

**原因：** 文档中使用的字体在对方电脑上未安装

**解决方法：**
- 将文字转换为曲线（嵌入）
- 导出为 PDF
- 使用常见字体

#### 3.3.1.7.4 表格跨页断开

**解决方法：**
- 选中表格，右键选择“表格属性”
- 勾选“允许跨页断行”
- 或勾选“在各页顶端以标题行形式重复”

#### 3.3.1.7.5 行距无法调整

**常见原因：**
- 段落设置了固定行距
- 文档中有图形或表格影响

**解决方法：**
- 检查段落设置中的行距选项
- 清除格式后重新设置

### 3.3.1.8 进阶技巧与常见问题

#### 3.3.1.8.1 查找与替换高级用法

**基本查找：**
- 打开查找框（Ctrl+F）
- 输入要查找的内容
- 使用“查找下一个”定位

**替换功能：**
- 打开替换框（Ctrl+H）
- 输入查找内容和替换内容
- 可选择“替换”或“全部替换”

**高级选项：**
- 区分大小写
- 全字匹配
- 使用通配符
- 格式搜索

#### 3.3.1.8.2 样式与格式刷

**样式的作用：**
- 快速统一格式
- 方便修改全局样式
- 生成目录的基础

**格式刷使用：**
- 单击格式刷：复制一次格式
- 双击格式刷：可多次使用
- 清除格式：选中后点击清除格式

#### 3.3.1.8.3 目录自动生成

**创建目录：**
1. 为标题应用样式（标题1、标题2等）
2. 插入目录
3. 选择目录样式
4. 更新目录

**更新目录：**
- 右键目录选择“更新域”
- 选择“更新整个目录”

#### 3.3.1.8.4 批注与修订

**批注：**
- 选中内容，插入批注
- 可回复、删除批注
- 适合提出建议或疑问

**修订：**
- 开启修订模式
- 所有修改会被记录
- 可接受或拒绝修改

#### 3.3.1.8.5 分栏排版

**使用场景：**
- 书籍、杂志排版
- 简历制作
- 通知文书

**设置方法：**
1. 选中要分栏的内容
2. 点击“布局”->“分栏”
3. 选择栏数
4. 可设置分隔线

#### 3.3.1.8.6 首字下沉

**设置方法：**
1. 光标放在段落中
2. 点击“插入”->“首字下沉”
3. 选择下沉方式
4. 设置字体、下沉行数

#### 3.3.1.8.7 脚注与尾注

**脚注：**
- 页面底部注释
- 用于解释说明

**尾注：**
- 文档末尾注释
- 用于引用来源

**插入方法：**
1. 光标放在要添加注释的位置
2. 点击“引用”->“插入脚注”或“插入尾注”

#### 3.3.1.8.8 邮件合并

**用途：**
- 批量生成邀请函
- 制作信封标签
- 生成成绩单

**基本步骤：**
1. 准备数据源（Excel或Word表格）
2. 插入合并域
3. 预览结果
4. 完成合并

#### 3.3.1.8.9 文档比较与合并

**比较文档：**
- 审阅->比较
- 显示两家文档的差异

**合并文档：**
- 审阅->合并
- 将多个文档合并为一个

#### 3.3.1.8.10 快捷键汇总

**文字编辑：**
- Ctrl+B：加粗
- Ctrl+I：斜体
- Ctrl+U：下划线
- Ctrl+Shift+>：增大字号
- Ctrl+Shift+<：减小字号

**段落格式：**
- Ctrl+L：左对齐
- Ctrl+E：居中
- Ctrl+R：右对齐
- Ctrl+J：两端对齐

**文件操作：**
- Ctrl+N：新建文档
