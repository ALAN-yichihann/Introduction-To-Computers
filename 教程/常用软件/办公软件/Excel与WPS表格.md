## 3.3.2 Excel 与 WPS 表格

![AI辅助](../../../图片/ai.svg)

Excel 和 WPS 表格属于“电子表格软件”，主要用于记录数据、制作成绩表、统计数量、计算总和、制作值班表等。

很多初学者第一次看到它会觉得格子很多、很复杂，但只要先认识“行、列、单元格”，就能很快上手。

### 3.3.2.1 认识行、列和单元格

![简单](../../../图片/easy.svg)

- 横着的是“行”，通常用数字编号，如第 `1` 行、第 `2` 行。
- 竖着的是“列”，通常用字母编号，如 `A` 列、`B` 列、`C` 列。
- 某一个具体的格子叫“单元格”，例如 `A1`、`B2`、`C8`。

例如：

- `A1` 表示第 `A` 列、第 `1` 行
- `C3` 表示第 `C` 列、第 `3` 行

如果别人说“把姓名填到 `A` 列”，意思就是让你把姓名填在第一列。

**行列编号系统详解：**  
Excel 的行列编号系统是电子表格的基础。行号从 1 开始，可达 1,048,576 行；列标从 A 开始，至 XFD（16,384 列）。单元格地址如 `A1` 是绝对的。

**名称框的使用：**  
名称框显示当前单元格地址，可输入地址快速定位，或定义命名区域简化公式，如 =SUM(销售额)。

### 3.3.2.2 输入和整理数据

![简单](../../../图片/easy.svg)

单击一个单元格后，就可以直接输入内容。

你可以输入：

- 文字，例如姓名、班级、备注
- 数字，例如成绩、价格、数量
- 日期，例如 `2026/03/28`

输完后按 `Enter`，通常会跳到下一行。

为了让表格更清楚，建议：

- 第一行先写表头，例如“姓名”“语文”“数学”“总分”
- 同一列尽量保持同一种类型的数据
- 不要把一大段解释文字塞进成绩表这类严肃表格里

**数据类型详解：**

| 数据类型 | 特点 | 用途 |
|----------|------|------|
| 文本型 | 包括汉字、字母、符号等，可以包含数字但被当作文本处理 | 姓名、备注等 |
| 数值型 | 包括整数、小数、分数，可以进行数学运算 | 成绩、价格、数量等 |
| 日期时间型 | 内部存储为数字，可以进行日期计算，自定义显示格式 | 日期、时间 |
| 逻辑型 | TRUE 和 FALSE | 条件判断 |
| 错误值 | #DIV/0!（除数为零）、#N/A（数据不可用）、#NAME?（无法识别名称）、#VALUE!（错误的值类型） | 错误提示 |

**数据输入技巧：**  
- **批量输入：** 选中多个单元格后输入，按 `Ctrl`+`Enter` 确认。  
- **自动换行：** 在单元格内按 `Alt`+`Enter` 强制换行。  
- **填充柄使用：** 拖动单元格右下角的小方块快速复制或填充。  
- **日期输入：** 直接输入格式或使用 `Ctrl`+`;` 输入当前日期，`Ctrl`+`Shift`+`;` 输入时间。

### 3.3.2.3 最常见的计算公式

![简单](../../../图片/easy.svg)

表格软件最重要的意义之一，就是可以自动计算。

公式一定要以 `=` 开头。常见公式例如：

- `=A1+B1`：把 `A1` 和 `B1` 的内容相加
- `=SUM(B2:B10)`：把 `B2` 到 `B10` 这一段全部求和
- `=AVERAGE(B2:B10)`：求平均数

如果你做的是成绩表，最常用的通常就是 `SUM` 和 `AVERAGE`。

>[!IMPORTANT]
>如果你输入的内容没有以 `=` 开头，那么软件只会把它当普通文字，不会计算。

**公式基础：**  
公式的组成包括等号开头、操作数（单元格引用、常量、函数）和运算符。单元格引用有相对（`A1`）、绝对（`$A$1`）和混合（`A$1`）类型。运算符优先级：括号 > 负号 > 百分比 > 乘除 > 加减。

**常用数学函数：**

| 函数 | 语法 | 示例/说明 |
|------|------|-----------|
| SUM | =SUM(数字1, 数字2, ...) 或 =SUM(区域) | =SUM(A1:A10) 求 A1 到 A10 的和 |
| AVERAGE | =AVERAGE(区域) | 计算区域中所有数值的平均值，自动忽略空白单元格和文本 |
| MAX/MIN | =MAX(区域) / =MIN(区域) | 返回区域中的最大值/最小值 |
| COUNT | =COUNT(区域) | 统计区域中数值单元格的个数 |
| COUNTA | =COUNTA(区域) | 统计非空单元格个数 |
| COUNTBLANK | =COUNTBLANK(区域) | 统计空白单元格个数 |

**条件函数：**

| 函数 | 语法 | 示例/说明 |
|------|------|-----------|
| IF | =IF(条件, 条件成立时的值, 条件不成立时的值) | =IF(A1>=60, "及格", "不及格") |
| SUMIF | =SUMIF(条件区域, 条件, 求和区域) | =SUMIF(B:B, ">60", B:B) 计算大于60的总数 |
| COUNTIF | =COUNTIF(区域, 条件) | =COUNTIF(A:A, "男") 统计男性人数 |
| IFERROR | =IFERROR(公式, 错误时的值) | 用于处理公式错误，避免显示错误值 |

### 3.3.2.4 调整列宽、对齐和边框

![简单](../../../图片/easy.svg)


新手最常碰到的问题之一，就是内容显示不全或表格不够美观。

**常用调整方法：**
- **拉宽列宽**：把鼠标移到两列标题之间拖动，适当调整宽度。
- **自动适应**：双击列标题分隔线，自动适应内容宽度。
- **调整对齐方式**：让内容居中、左对齐或右对齐，提升可读性。
- **添加边框**：让表格看起来更整齐，常用于打印或正式文档。
- **设置数字格式**：如保留两位小数、显示货币符号、日期格式等。
- **合并单元格**：适合做表头、分类汇总等。
- **填充颜色**：区分不同区域或突出重点。

**应用场景举例：**
- 做成绩表、值班表、物品清单、统计报表等。
- 通过合理排版让表格更清晰美观。

### 3.3.2.5 自动填充

![简单](../../../图片/easy.svg)

如果你已经输入了一个单元格，很多时候可以拖动右下角的小方块，让它自动填充。

**常见用途有：**
- 复制同样的内容到下面几行
- 自动生成序号（如1、2、3……）
- 让日期、星期等继续往下排
- 自动补全公式、批量填充数据

**进阶技巧：**
- 输入“1”“2”，选中两个单元格再拖动，可以自动生成递增序号
- 输入“星期一”，下拉可自动补全“星期二”“星期三”……

这是做表格时非常常用、也非常省时间的技巧。

**填充系列详解：**  
- **数值系列：** 起始值+拖动复制相同值；两个值+拖动等差填充；按Ctrl切换复制/填充。  
- **日期系列：** 日期+拖动按天递增；+Ctrl复制相同日期；月份/季度按月/季度递增。  
- **文本系列：** 纯文本复制；带数字的文本数字递增；自定义列表按预设序列。

**填充选项：** 拖动后出现选项：复制单元格、仅填充序列、填充格式、不带格式填充。

**快速填充（Ctrl+E）：** Excel 2013+ 根据示例自动识别填充规律。
- 适合提取、合并、转换数据
- 按 Ctrl+E 或点击“数据”->“快速填充”

### 3.3.2.6 排序、筛选和打印

![简单](../../../图片/easy.svg)


当数据变多之后，排序和筛选就会很有用。

**排序功能：**
- 可以按分数、日期、姓名等从高到低或从低到高排列
- 支持多列排序（如先按班级，再按分数）

**筛选功能：**
- 只看某个班级、某种类别的数据
- 可以快速隐藏不需要的数据，便于分析

**查找与替换：**
- 用于快速定位、批量修改内容

**打印注意事项：**
- 表格太宽时，可能需要横向页面
- 打印前先看预览，确认没有被截断
- 表头最好放在最上面，方便阅读
- 可设置“打印标题行”，每页都显示表头
- 可设置页码、页眉页脚

**排序功能详解：**

| 类型 | 方法 |
|------|------|
| 单列排序 | 选中要排序的列，点击“数据”->“排序”，选择升序或降序 |
| 多列排序 | 添加多个排序条件，设置主要关键字、次要关键字等，可设置自定义排序顺序 |
| 自定义排序 | 按颜色排序，按单元格图标排序，按自定义序列排序 |

**筛选功能详解：**

| 类型 | 方法 |
|------|------|
| 启用筛选 | 选中数据区域，点击“数据”->“筛选”，或按 Ctrl+Shift+L |
| 筛选操作 | 点击列标题的下拉箭头，勾选要显示的项目，可按文本筛选或数字筛选 |
| 高级筛选 | 可设置复杂条件，可提取不重复记录，可在另一位置显示结果 |

**打印设置：**

| 类型 | 方法 |
|------|------|
| 页面设置 | 方向：纵向或横向；缩放：调整打印比例；页边距：设置上下左右边距 |
| 打印区域 | 选中区域，点击“设置打印区域”，可设置多个打印区域 |
| 标题行 | 页面布局->打印标题，设置顶端标题行，设置左端标题列 |
| 打印预览 | 文件->打印->预览，检查分页位置，确认格式正确 |

### 3.3.2.7 初学者常见问题

![简单](../../../图片/easy.svg)

**常见问题及解决方法：**

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 为什么不自动计算 | 公式前面忘了加 =；公式写错了单元格位置；把数字输入成了文字 | 先从最简单的 =A1+B1、=SUM(...) 开始练习 |
| 为什么表格打印出来很奇怪 | 页面方向不对；列太宽，超出纸张范围；打印区域设置不正确；打印前没有先看预览 | 打印前务必使用“打印预览”，调整列宽、页面方向、边距，设置好打印区域和标题行 |
| 公式不计算显示为文本 | 前面缺少等号；单元格格式为文本；公式前面有空格 | 检查是否以 = 开头；更改单元格格式为常规；重新输入公式 |
| 循环引用问题 | 公式引用了自己所在的单元格，可能导致计算错误 | 检查公式中的单元格引用；使用“文件”->“选项”->“公式”启用迭代计算（谨慎使用） |
| 日期显示为数字 | 日期本质是数字，只是显示格式不同 | 设置单元格格式为日期；或使用 TEXT 函数转换显示格式 |
| #VALUE! 等错误 | #DIV/0!：除数为零或空单元格；#NAME?：无法识别的函数名或名称；#REF!：引用了已删除的单元格；#VALUE!：数据类型不匹配；#N/A：查找函数找不到匹配值 | 检查公式引用的单元格；使用 IFERROR 捕获错误；检查函数参数是否正确 |

### 3.3.2.8 进阶技巧与常见问题

#### 3.3.2.8.1 冻结窗格

当表格很长时，可以"冻结首行"或"冻结首列"，方便滚动时始终看到表头。

**冻结首行：**
- 视图->冻结窗格->冻结首行
- 滚动时第一行始终可见

**冻结首列：**
- 视图->冻结窗格->冻结首列
- 滚动时第一列始终可见

**冻结拆分窗格：**
- 选中某个单元格
- 点击冻结窗格->冻结窗格
- 该单元格左上区域会被冻结

**取消冻结：**
- 再次点击冻结窗格->取消冻结

#### 3.3.2.8.2 条件格式

可以根据内容自动变色，如分数低于60分自动标红。

**突出显示规则：**
- 大于、小于、介于
- 等于、特定文本
- 发生日期

**项目选取规则：**
- 前10项/后10项
- 前10%/后10%
- 高于/低于平均值

**数据条：**
- 用颜色条显示数值大小
- 适合直观比较数据

**色阶：**
- 用颜色渐变表示数值大小
- 适合显示数据分布

**图标集：**
- 用图标表示数据状态
- 适合仪表盘类展示

**自定义规则：**
- 使用公式确定要设置的单元格
- 可实现复杂条件格式

#### 3.3.2.8.3 数据透视表

适合做复杂的数据统计和分析，自动生成汇总报表。

**创建数据透视表：**
1. 选中数据区域
2. 插入->数据透视表
3. 选择放置位置
4. 拖动字段到相应区域

**字段区域：**
- 行：将字段显示为行标题
- 列：将字段显示为列标题
- 值：对数据进行汇总计算
- 筛选：可筛选数据

**值汇总方式：**
- 求和、计数、平均值
- 最大值、最小值
- 标准偏差、方差

**值显示方式：**
- 常规、百分比
- 父行总计百分比
- 累计求和

**切片器：**
- 可视化筛选器
- 方便交互式筛选

**推荐的数据透视表：**
- 快速分析数据
- 自动生成推荐表格

#### 3.3.2.8.4 图表制作

**创建图表：**
1. 选中数据
2. 插入->选择图表类型
3. 调整图表元素

**常用图表类型：**
- 柱形图：比较不同类别

- 饼图：显示占比
- 条形图：横向比较
- 散点图：显示相关性
- 面积图：强调总量

**图表元素：**
- 标题、图例

**图表样式：**
- 快速布局

| 格式 | `Ctrl`+`Shift`+`%` | 百分比格式 |
| 格式 | `Ctrl`+`Shift`+`#` | 日期格式 |
| 其他 | `F4` | 切换引用类型 |
| 其他 | `Alt`+`=` | 自动求和 |
| 其他 | `Ctrl`+`;` | 输入当前日期 |
| 其他 | `Ctrl`+`Shift`+`;` | 输入当前时间 |
