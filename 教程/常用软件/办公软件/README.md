## 3.3 办公软件

办公软件，是用于处理文档、表格、演示文稿等内容的软件。写通知、写作文、做表格、做汇报、做简历，这些事情通常都离不开办公软件。

对于 Windows 10 用户来说，最常见的办公软件主要有两类：

- Microsoft Office：常见组件有 Word、Excel、PowerPoint，很多学校和单位默认使用这一套。
- WPS Office：国内很常见，常见组件有 WPS 文字、WPS 表格、WPS 演示，和 Office 的操作习惯比较接近。

>[!TIP]
>如果你只是想先学会“怎么用办公软件”，那么学会其中一套就够了。因为这两套软件的基本思路很接近：写文章、做表格、做幻灯片的方法都差不多。

### 常见文件类型

![简单](../../../图片/easy.svg)

办公软件中最常见的几种文件格式如下：

文件格式 | 含义 | 常见用途
| - | - | - |
`.docx` | 文档文件 | 文章、通知、简历、报告
`.xlsx` | 表格文件 | 成绩表、值班表、统计表
`.pptx` | 演示文稿文件 | 课件、展示、答辩
`.pdf` | 便携式文档 | 用于阅读、打印、发送，版式较稳定

>[!IMPORTANT]
>`.pdf` 通常更适合“发给别人看”，而不是自己继续大幅修改；`.docx`、`.xlsx`、`.pptx` 更适合继续编辑。

### 通用操作

![简单](../../../图片/easy.svg)

虽然文字、表格、幻灯片看起来不同，但它们有很多共同的基本操作。

#### 新建文件

打开软件后，通常会看到“新建”或“空白文档”“空白工作簿”“空白演示文稿”等选项。

- 写文章时，选择“空白文档”。
- 做表格时，选择“空白工作簿”。
- 做幻灯片时，选择“空白演示文稿”。

有些软件还会提供模板，例如简历模板、课程表模板、工作总结模板。如果你只是想快速做一个像样的成品，也可以直接使用模板。

#### 打开已有文件

有两种常见方式：

- 在软件里点击“打开”，再找到对应文件。
- 在文件资源管理器中直接双击该文件，系统会自动调用对应软件打开。

如果双击文件后打开的软件不对，可以在文件上右击，选择“打开方式”后换一个软件。

#### 保存与另存为

保存，是初学者最应该养成的习惯。

- `Ctrl + S`：保存当前文件。
- “另存为”：把文件保存到新的位置、起新的名字，或者换一种格式保存。

举例：

- 你正在写一篇通知，临时保存进度，用“保存”即可。
- 你想把“作业模板”另存为“张三-作业1”，就应该用“另存为”。
- 你想把文档导出成 PDF，也通常要通过“另存为”或“导出”来完成。

>[!IMPORTANT]
>不要等内容全部写完才第一次保存。建议新建文件后，先立刻保存一次，给它起一个明确的文件名。

#### 如何选择保存位置

Windows 10 中常见的保存位置有：

- 桌面：最容易找到，但文件多了会很乱。
- 文档：适合保存文稿、表格、课件。
- 下载：适合临时存放从网上获取的文件，但不适合长期堆积重要内容。
- U 盘：适合拷贝到其他电脑，但注意不要突然拔出。

对于新手来说，建议专门建立一个文件夹，例如“办公练习”“学校作业”“工作文件”，以后会更好找。

#### 导出 PDF

很多时候，我们并不希望别人继续修改文件，而是希望对方看到固定不乱的版式。这时就应该导出 PDF。

常见做法是：

1. 点击“文件”。
2. 找到“导出”或“另存为”。
3. 选择格式为 `PDF`。
4. 选择保存位置并保存。

导出后的 PDF 更适合：

- 打印
- 发给老师、同学、同事查看
- 提交简历、申请表、作业成品

#### 打印

打印的基本路径通常为“文件” -> “打印”。

打印前要重点看 4 件事：

- 打印机是否选对
- 打印页数是否选对
- 单面还是双面
- 纸张方向是否正确（纵向或横向）

如果是表格太宽，打印前通常还要先检查是否需要把页面改成横向。

### 发送文件前的注意事项

![简单](../../../图片/easy.svg)

办公文件在发送给别人之前，建议检查以下几项：

1. 文件是否已经保存
2. 文件名是否清楚，例如“高一（3）班值日表”“张三-求职简历”
3. 是否发成了合适的格式
4. 图片和内容是否完整显示

常见建议如下：

- 还要继续改，就发源文件，例如 `.docx`、`.xlsx`、`.pptx`
- 只想让别人阅读或打印，就优先发 `PDF`
- 发给别人前，自己先重新打开一次文件检查最稳妥

### 小结

![简单](../../../图片/easy.svg)

办公软件并不神秘。对初学者来说，最重要的不是一下子学会所有功能，而是先掌握以下几件事情：

1. 分清文档、表格、演示文稿三类软件分别是干什么的
2. 学会新建、打开、保存、另存为、导出 PDF
3. 养成随手保存和规范命名文件的习惯
4. 根据自己的需要，继续阅读 Word、Excel、PowerPoint 的分支教程
