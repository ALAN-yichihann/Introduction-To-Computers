常用软件是指在日常使用电脑时经常接触到的一类程序，例如浏览器、解压缩软件、办公软件等。它们并不一定是系统自带的，但往往是学习、办公、上网和传文件时离不开的工具。

该部分旨在帮助第一次系统接触电脑的读者认识这些软件，并学会最基础、最常见的使用方法。

为贴合大众，本部分基于 Windows 10 进行编写。

在学习此部分之前，确保你对于文件系统已经有了基本的了解。这意味着，请确保已经学习过系统部分的2.1~2.8节。考虑到你已经对电脑有了初步的了解，本部分的图片量将会减少。同时，将采用类比的方式，例如一类软件中只介绍一个软件，因为你已经对电脑有了初步的自主学习能力。
