## 1.1 杂项

这里存放一些不好归类但对初学者很有帮助的硬件知识。

### 1.1.1 强制关机与正常关机

![简单](../../图片/easy.svg)

- **正常关机**：通过操作系统菜单选择“关机”或使用快捷键，让系统按顺序关闭程序、保存数据、断开硬件。
- **强制关机**：长按电脑电源键 5 秒或更长时间，直接切断电源，使电脑立即断电。

正常关机是最安全的关机方式。强制关机通常用于系统完全无响应、鼠标键盘不能操作时作为最后手段。

> [!IMPORTANT]
> 强制关机可能导致未保存的数据丢失、硬盘文件系统损坏，甚至对硬件造成应力损伤。只在无法正常关机时使用。


### 1.1.2 开机顺序与 BIOS/UEFI

![适中](../../图片/medium.svg)

开机过程不仅仅是按下电源键。电脑启动时会经历以下几个阶段：

1. **通电自检（POST）**
   - 主板上电后，会检测CPU、内存、显卡等是否正常。
   - 如果关键硬件异常，可能出现蜂鸣声、无法启动或显示错误代码。

2. **加载 BIOS/UEFI**
   - BIOS/UEFI 是主板上的固件，负责初始化硬件、检测设备、查找系统盘。
   - 现代电脑通常使用 UEFI，支持更大的硬盘、更快启动和图形界面设置。

3. **读取启动设备**
   - 电脑会按照“启动顺序”查找可启动设备，例如SSD、机械硬盘、USB优盘、DVD光驱。
   - 如果顺序设置不当，可能会出现“找不到系统”或从U盘错误启动的情况。

4. **加载操作系统**
   - 找到系统后，启动程序将操作系统内核加载到内存中，随后进入登录界面。

如果你想改变启动顺序，可进入 BIOS/UEFI 设置界面，通常在开机时按 `Del`、`F2`、`F10` 等键进入。


### 1.1.3 安全重启与恢复模式

![简单](../../图片/easy.svg)

- **重启**：相当于先正常关机再重新开机。它会关闭程序、释放内存、重新加载驱动。
- **强制重启**：当系统卡住时，可先强制关机再开机，但同样不推荐频繁使用。
- **恢复模式/安全模式**：在出现启动失败或系统问题时，可进入系统的安全模式或恢复选项，排查驱动、硬件兼容性问题。

在 Windows 中，按住 `Shift` 键点击“重启”，可进入高级启动菜单；在 macOS 中，按住开机键或 `Command + R` 可进入恢复模式。


### 1.1.4 常见硬件报警与指示灯

![适中](../../图片/medium.svg)

很多电脑通过蜂鸣声、指示灯或故障代码来提醒硬件问题。常见情况包括：

- **蜂鸣声**：通常来自主板，用于提示内存、显卡、CPU 或显示器等故障。不同的蜂鸣组合代表不同问题。
- **电源指示灯闪烁或不亮**：可能是电源故障、主板异常、电源按钮或电源线问题。
- **硬盘访问灯常亮或闪烁异常**：有时表示硬盘正在大量读写；如果伴随异常噪音或系统卡顿，可能意味着硬盘故障。
- **风扇不停转或停转**：持续高速转动可能是散热不足，停转通常说明风扇故障或电源问题。

遇到硬件报警时，先不要盲目拆机。记录报警方式，查阅电脑说明书或主板手册，必要时联系专业人员。


### 1.1.5 硬件升级与兼容性

![简单](../../图片/easy.svg)

硬件升级不是“买最贵的就行”，而是“买适合自己的”。以下是常见的升级判断：

- **内存升级**：如果电脑经常卡顿且内存占用接近 90%，可以考虑增加内存容量；
- **硬盘升级**：系统盘最好使用 SSD，数据盘则可以配合大容量机械硬盘；
- **显卡升级**：适合游戏、视频剪辑和3D应用，但要保证电源和机箱空间足够；
- **电源升级**：添加高性能显卡或更多硬盘时，需确认电源瓦数和接口是否足够。

升级前，请先确认主板支持哪些硬件：

- CPU 插槽类型与兼容型号；
- 内存类型（DDR4、DDR5）和最大支持容量；
- 硬盘接口（SATA、M.2）和可用插槽；
- 显卡插槽类型（PCIe x16）和机箱空间。


### 1.1.6 电源与散热基础

![适中](../../图片/medium.svg)

电源和散热是硬件稳定运行的基础。初学者可以从这几个点理解：

- **电源供应器（PSU）**：为整机提供电力。瓦数不足会导致系统不稳定、重启或无法启动。
- **散热风扇**：CPU、显卡、机箱风扇负责把热量排出机箱。风扇堵灰会导致温度升高。
- **散热器/散热片**：安装在 CPU、显卡和某些芯片上，帮助将热量传导出来。
- **导热硅脂**：在 CPU 和散热器之间涂抹，用于填补空隙，提高热传导效率。

保持机箱内部通风、清理灰尘、及时更换老化风扇，是硬件寿命延长的关键。


### 1.1.7 外设接口与常见连接方式

![简单](../../图片/easy.svg)

电脑与外部设备之间的连接方式很多。常见接口包括：

- **USB**：最常见的外设接口，支持鼠标、键盘、U盘、打印机、手机充电等；
- **HDMI / DisplayPort / VGA**：用于连接显示器或电视；
- **以太网口（RJ45）**：用于有线网络连接；
- **音频接口**：耳机、麦克风、音箱使用；
- **Thunderbolt / USB-C**：现代高速接口，可用于数据传输、充电和外接显示器。

不同接口速度和功能不同。购买或连接外设前，先确认电脑和设备支持的接口类型。


### 1.1.8 硬件保养与安全使用

![适中](../../图片/medium.svg)

硬件保养并不复杂，关键是做好这些基本习惯：

- **避免潮湿和高温**：水汽和高温会加速元件老化；
- **避免频繁插拔敏感接口**：例如内存、显卡、硬盘等，插拔时要小心；
- **防静电**：拆装硬件前先接触金属机箱，释放静电；
- **定期备份数据**：硬件不是永久可靠，重要资料需要多处保存；
- **使用正版操作系统和驱动程序**：错误驱动可能导致硬件异常。

> [!TIP]
> 即使你不是发烧友，了解一些硬件基础也能让你在遇到问题时更容易判断该继续使用、该维修，还是需要更换。
