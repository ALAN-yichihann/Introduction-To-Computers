## 4.2 网络使用

![简单](../../图片/easy.svg)

本节详细介绍日常网络连接、浏览器使用、下载/上传、网络设置以及网络问题排查的方法。我们会把常见操作拆分成更具体的步骤，并结合截图示例帮助你理解。

### 4.2.1 连接家庭 Wi-Fi

#### 4.2.1.1 什么是 SSID 和密码

Wi-Fi 的名称叫做 SSID，它是无线网络的“名字”。例如你的家庭 Wi-Fi 可能叫 `Home-1234`、`Xiaomi-5G` 或 `TP-Link_5G`。

密码则是连接该 Wi-Fi 时需要输入的口令。家庭路由器背面通常会有默认密码，或者由网络管理员设置。

> 提示：如果你不确定密码，先问家里会使用路由器的人，或者查看路由器底部标签。

#### 4.2.1.2 连接过程

1. 单击任务栏右下角的网络图标，通常是一个 Wi-Fi 扇形图标；
2. 在可用网络列表中找到你的家庭 Wi-Fi；
3. 单击该名称，然后点击 **“连接”**；
4. 在弹出的输入框中输入 Wi-Fi 密码；
5. 点击 **“下一步”** 或 **“连接”**，等待系统完成连接。

<img src='../../图片/xt-net-wifi.svg' alt="Wi-Fi 连接" height=300pt>

如果你看到 **“已连接”**，说明电脑已经和路由器建立了连接。如果显示 **“已连接、受限”**，说明你已经连接路由器，但还没有成功上网。

#### 4.2.1.3 2.4GHz 与 5GHz

很多家庭路由器会同时提供 2.4GHz 和 5GHz 两个频段。它们的区别是：

- 2.4GHz：穿墙能力强，覆盖范围大，但网速较低；
- 5GHz：速度更快，但信号穿透力弱，适合靠近路由器的设备。

如果你离路由器较近，优先选择 5GHz；如果离路由器较远，可以选择 2.4GHz。

#### 4.2.1.4 选择网络类型

连接 Wi-Fi 后，Windows 可能会问你这是否是“私有网络”或“公共网络”。

- 私有网络：适合家庭或办公室，允许设备之间共享文件和打印机；
- 公共网络：适合咖啡店、机场等，安全性更高，但限制设备间共享。

在家里建议选择 **“私有网络”**，避免误将家庭网络设置成公共网络。

#### 4.2.1.5 忘记网络

如果你输入密码后还是无法连接，先把该网络忘记，然后重新连接：

- 打开 **“设置” > “网络和 Internet” > “Wi-Fi”**；
- 点击 **“管理已知网络”**；
- 选择当前家庭 Wi-Fi，点击 **“忘记”**；
- 重新选择该网络并输入正确密码。

这样可以清除旧密码和过期配置，重新建立连接。

#### 4.2.1.6 Wi-Fi 信号弱怎么办

- 尽量靠近路由器；
- 避免把路由器放在柜子里、金属物品旁或玻璃窗后；
- 如果家里有多间房，可以考虑在另一间房放一个无线中继器或 Mesh 路由器。

### 4.2.2 使用移动网络

部分笔记本和平板支持插入 SIM 卡，可以通过移动网络上网。移动网络比 Wi-Fi 更适合在户外、出差或没有无线网络时使用。

#### 4.2.2.1 打开移动网络

1. 打开 **“设置” > “网络和 Internet” > “移动网络”**；
2. 确认 SIM 卡已正确插入；
3. 开启 **“移动数据”**；
4. 等待系统显示运营商名称和信号强度。

<img src='../../图片/xt-net-mobile.svg' alt="移动网络设置" height=300pt>

#### 4.2.2.2 常见问题与解决

- 找不到 SIM 卡：请重新插拔 SIM 卡，确认金属触点干净；
- 无信号：移动到窗户附近或楼层较高处；
- 连接但无法上网：有时需要设置 APN，或者运营商套餐流量已经用完。

如果你不确定套餐细节，建议联系该运营商客服确认流量、欠费与 APN 设置。

#### 4.2.2.3 流量使用建议

移动网络通常按流量计费。建议：

- 不要在移动网络下大量下载视频或文件；
- 如果设备支持，可将移动网络设置为“计量连接”，减少后台更新；
- 尽量使用 Wi-Fi 下载大文件。

### 4.2.3 使用公共 Wi-Fi

公共 Wi-Fi 方便，但风险更大。连接公共 Wi-Fi 时要格外小心。

#### 4.2.3.1 连接公共 Wi-Fi 的正确步骤

1. 选择可信的公共 Wi-Fi 名称；
2. 连接后打开浏览器；
3. 等待自动跳转到登录页面；
4. 按照提示输入验证码、手机号或点击同意协议。

<img src='../../图片/xt-net-public.svg' alt="公共 Wi-Fi 登录" height=300pt>

如果浏览器没有自动跳转，请手动输入 `www.baidu.com`、`www.qq.com` 等常用网站，通常会自动显示登录页面。

#### 4.2.3.2 公共 Wi-Fi 的安全提醒

- 避免在公共 Wi-Fi 上登录银行、支付、邮箱等敏感账号；
- 优先使用 HTTPS 网站，地址栏出现锁形图标更安全；
- 不要连接来路不明的公共网络。

>[!TIP]
>如果你必须在公共 Wi-Fi 上处理重要事务，尽量使用自己的移动网络或 VPN。

### 4.2.4 浏览网页与搜索

浏览器是你进入互联网的窗口。熟悉浏览器基本功能，可以让你更安全、更高效地上网。

#### 4.2.4.1 地址栏与搜索

浏览器的地址栏既可以输入网址，也可以输入搜索词：

- 输入网址：例如 `www.baidu.com`；
- 输入搜索词：例如 `天气`、`作业答案`；
- 输入浏览器特殊地址：例如 `edge://settings` 或 `chrome://dino`（仅在对应浏览器中有效）。

<img src='../../图片/xt-net-browser.svg' alt="浏览器地址栏" height=300pt>

如果你想了解更多浏览器的使用技巧和安全设置，参见[3.1 浏览器](../常用软件/浏览器.md#311-浏览器的选择)。

#### 4.2.4.2 标签页与历史记录

标签页允许你同时打开多个网页。建议：

- 同一主题保留在同一窗口；
- 不再使用的标签页及时关闭；
- 如果误关闭标签页，可在历史记录中恢复。

浏览器历史记录存储你访问过的网站。你可以通过浏览器右上角的菜单打开 **“历史记录”**，找到最近访问过的页面。

#### 4.2.4.3 书签和收藏夹

如果你经常访问某些网站，可以添加书签：

- 在地址栏右侧点击星形按钮；
- 在弹出的对话框中选择保存位置；
- 以后可以从收藏夹中快速打开。

#### 4.2.4.4 私密窗口与隐私模式

浏览器通常提供隐私窗口或无痕模式。它不会保存浏览历史、Cookies 或缓存。适合：

- 公共电脑；
- 不想让别人看到浏览历史；
- 尝试临时访问一个网站。

但它不能隐藏你对运营商、学校或公司网络管理员的访问记录。

#### 4.2.4.5 页面加载慢时的处理

- 尝试刷新页面；
- 关闭无关标签页；
- 检查当前网络是否存在其他占用，例如下载、视频或游戏；
- 如仍慢，尝试打开其他网站，看是否全网慢。

### 4.2.5 下载与上传

下载和上传是最常见的网络操作，理解流程和安全规则很重要。

#### 4.2.5.1 下载文件

1. 点击网页上的下载按钮；
2. 选择保存位置，通常是 **“下载”** 文件夹；
3. 等待浏览器完成下载；
4. 点击下载栏中的文件名打开，或在文件资源管理器中查看。

<img src='../../图片/xt-net-download.svg' alt="下载文件" height=300pt>

如果找不到下载的文件：

- 打开文件资源管理器；
- 进入 **“下载”** 文件夹；
- 或在浏览器下载列表中点击 **“显示文件夹”**。

#### 4.2.5.2 下载安全原则

- 只从官方网站或可信渠道下载软件；
- 避免下载 `.exe`、`.msi` 等可执行文件，除非你确定来源；
- 对于压缩文件（`.zip`、`.rar`），下载后用解压软件打开，详见[3.2 解压缩软件](../常用软件/解压缩软件.md)。

#### 4.2.5.3 上传文件

上传文件时，请确认：

- 选择了正确的文件；
- 文件大小是否符合限制；
- 当前网络是否稳定。

常见上传场景：

- 邮件附件；
- 网盘；
- 社交软件；
- 在线表单提交。

### 4.2.6 常见网络应用

网络应用包括浏览器、邮箱、网盘、社交软件和在线视频服务。

- 邮件：用于收发电子邮件；
- 网盘：用于备份和共享文件；
- 社交软件：用于聊天和沟通；
- 视频平台：用于观看视频；
- 在线文档：用于多人协作编辑。

<img src='../../图片/xt-net-apps.svg' alt="网络应用示例" height=300pt>

这些应用大部分都需要登录账号。重要账号建议使用不同密码，并开启二步验证。

### 4.2.7 基本网络设置

当网络不稳定时，先查看系统设置，确认当前网络状态。

#### 4.2.7.1 查看当前网络

1. 打开 **“设置” > “网络和 Internet”**；
2. 查看当前连接类型：Wi-Fi、以太网、移动网络；
3. 点击当前网络，查看详细信息。

<img src='../../图片/xt-net-settings.svg' alt="网络设置" height=300pt>

#### 4.2.7.2 计量连接

如果你想控制流量使用，可以把当前网络设置为 **计量连接**。这样 Windows 会减少后台更新，适合移动网络或流量较少的 Wi-Fi。

#### 4.2.7.3 代理与 VPN

- 代理：某些学校或公司会要求通过代理上网，不要随意输入不明代理地址；
- VPN：可以加密你的连接，但要选择可信服务，不要使用未知来源的 VPN 客户端。

如果你不清楚这些设置，保持默认即可。

### 4.2.8 常见问题小结

#### 问题 1：为什么我的 Wi-Fi 不能连接？

- 确认密码是否正确；
- 确认路由器是否通电；
- 确认连接的是自己家的 Wi-Fi，而不是相似名称的陌生网络。

#### 问题 2：为什么已经连接但不能上网？

- 可能是路由器尚未连接互联网；
- 可能是运营商故障；
- 也可能是电脑还未分配到正确 IP 地址。

#### 问题 3：为什么网页打开很慢？

- 看是否有其他设备同时占用较大带宽；
- 检查是否使用的是公共 Wi-Fi；
- 重启路由器后再试。

#### 问题 4：为什么浏览器提示“网站不安全”？

- 当地址栏前没有锁形图标时，说明该网站没有启用 HTTPS；
- 避免在这种网站上输入账号、密码或银行卡信息。

#### 问题 5：为什么下载后文件打不开？

- 检查下载是否完整；
- 确认文件扩展名是否正确；
- 确认你是否已经安装对应程序。

#### 问题 6：公共 Wi-Fi 登录后仍然不能上网？

- 先打开浏览器访问 `www.baidu.com` 或 `http://neverssl.com`，这通常会强制跳转到登录页面；
- 如果登录页面打不开，尝试重新连接网络；
- 如果仍失败，可能是该公共 Wi-Fi 本身出现故障。

>[!TIP]
>判断网络问题时，先从电脑本身开始排查，然后再看路由器和运营商。逐步排查，问题更容易定位。
