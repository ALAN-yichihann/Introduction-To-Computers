## 4.1 网络基础知识

![简单](../../图片/easy.svg)

本章帮助你理解网络的基本结构、家庭网络如何连接互联网，以及常见网络术语的含义。理解这些概念，有助于你在后续学习网络使用和排查故障时更快上手。

### 4.1.1 网络的组成

网络由以下三个部分组成：

- 设备：电脑、手机、平板、路由器、交换机、光猫等；
- 连接介质：网线、Wi-Fi 无线信号、移动通信信号；
- 通信规则：称为协议，例如 TCP/IP、HTTP、DNS 等。

#### 4.1.1.1 什么是网络协议

协议是网络中不同设备之间交换数据时必须遵守的规则。就像人们之间交流需要先约定用哪种语言，网络设备之间也需要统一“说话方式”。

常见协议包括：

- TCP/IP：最基础的网络协议族，负责把数据拆分成小包并在网络中传输；
- HTTP：浏览网页时常用的协议，负责请求网页内容；
- HTTPS：HTTP 的加密版本，在浏览器中保护数据传输安全；
- DNS：把域名转换为 IP 地址；
- DHCP：自动分配 IP 地址给设备；
- 802.11：Wi-Fi 无线网络的标准协议；
- Ethernet：有线网络的协议标准，常见于网线连接。

这些协议分工不同，但它们共同协作，才能让你的电脑正常上网。

简单来说，网络就像一条信息高速公路，设备是车辆，连接介质是路面，通信规则则是交通规则。

### 4.1.2 局域网与广域网

- 局域网（LAN）：通常是家庭、办公室或学校内部的网络。它的覆盖范围较小，但速度快，适合局部设备互联。
- 广域网（WAN）：例如互联网，它将不同地区的局域网连接起来，范围大、跨越城市甚至国家。

在家里，路由器和电脑、手机、智能电视组成一个局域网。通过路由器连接到运营商提供的互联网后，就可以访问广域网中的网站和服务。

### 4.1.3 路由器、光猫与交换机

- 光猫（Modem）：光猫的作用是把运营商提供的光纤、电话线或电缆信号转换成家庭内部可以使用的网络信号。它通常位于宽带线路入口处，是连接家庭网络和运营商网络的第一关。
- 路由器：路由器负责把家庭内的多台设备连接到互联网，并在它们之间分配数据流量。它还负责管理不同设备之间的通信、分配 IP 地址、以及执行 NAT（网络地址转换）。现代家庭路由器大多数还内置 Wi-Fi 功能，提供无线网络。
- 交换机：交换机是一个网络设备，用来把多根网线连接在一起，扩展有线网络端口。它不会像路由器一样分配 IP 地址，而是把数据直接转发给目标设备。普通家庭如果只有几台有线设备，一般不需要交换机；办公室或机房则常用交换机来连接更多设备。

当你的电脑通过网线连接到路由器时，路由器会把电脑发送的数据转发到互联网；反过来，互联网返回的数据也会经过路由器，然后再传回你的电脑。

### 4.1.4 IP 地址与 DHCP

- IP 地址：每台联网设备都需要一个地址，便于识别和通信。IP 地址通常由四个数字组成，例如 `192.168.1.100`，称为 IPv4 地址。它类似于网络中的“门牌号”。
- DHCP：动态主机配置协议。DHCP 服务通常由路由器提供，它会自动把可用的 IP 地址分配给每台连接设备。这样你不需要手动设置地址，电脑和手机就可以自动联网。

家庭网络常用的是私有 IP 地址段，例如：

- `192.168.0.x`；
- `192.168.1.x`；
- `10.0.0.x`。

这类地址只能在本地网络内部使用，不会直接出现在互联网中。它们由路由器在局域网内部管理。

#### 静态 IP 与动态 IP

- 动态 IP：设备每次连接到网络时，路由器会自动分配一个地址。大多数家庭用户都使用动态 IP；
- 静态 IP：地址固定不变，通常用于服务器、打印机等需要长期稳定访问的设备。如果你不知道如何设置，建议保持默认的动态 IP。

### 4.1.5 DNS：域名和地址转换

DNS 是域名系统（Domain Name System）。它的作用是把我们熟悉的域名转换成网络能识别的 IP 地址。

例如：

- 你输入 `www.baidu.com`；
- DNS 服务器查找这个域名对应的 IP 地址；
- 浏览器再通过这个地址访问百度服务器。

可以把 DNS 想象成电话簿：你只需要记住名字，不需要记住电话号码。

#### 如果 DNS 失败怎么办？

如果 DNS 解析失败，浏览器就无法找到网站，即使你已经连接到网络。常见表现是“无法找到该网页”或“DNS 服务器未响应”。此时可以尝试：

- 检查网络是否连接；
- 重新启动路由器；
- 修改 DNS 为公共 DNS，例如 `114.114.114.114`、`8.8.8.8`。

### 4.1.6 NAT 与公网/私网

NAT（网络地址转换）是路由器将局域网内部地址转换成一个公网地址的技术。

家庭网络中，路由器会给每台电脑、手机分配私有 IP（例如 `192.168.1.100`），而互联网看到的是路由器的公网 IP。这样可以让多台设备共享一个公网地址上网。

- 私网地址：只能在家庭或办公室内部使用，不直接暴露在互联网；
- 公网地址：由运营商分配，互联网上的其他设备可以访问到这个地址。

这种方式的好处是节省公网地址，还能提高安全性，因为外部网络无法直接访问到内网中的每台设备。

### 4.1.7 常见网络术语

- 带宽：指单位时间内能传输的数据量，通常以 Mbps（兆比特/秒）或 Gbps（千兆比特/秒）表示。带宽大可以让下载、在线观看视频更顺畅，但并不代表网页一定更快。
- 延迟：也叫“ping 值”，是数据从你电脑发送到目标服务器再返回所需的时间，通常以毫秒（ms）为单位。延迟低时，游戏和视频通话更流畅；延迟高时，会感觉卡顿。
- MAC 地址：网卡的物理地址，每个网络适配器都有一个唯一的 MAC 地址。它通常由 12 位十六进制组成，例如 `00-1A-2B-3C-4D-5E`。MAC 地址用于局域网内部设备识别。
- Wi-Fi：无线局域网的通用名称，表示通过无线方式连接网络。Wi-Fi 信号通常有名称（SSID），你要连接时会在列表中看到这些名称。
- 运营商：提供互联网接入服务的公司。例如中国电信、中国移动、中国联通等。你家里用的宽带或移动流量，通常由运营商负责提供网络线路和信号。

#### 带宽与速率的区别

- 带宽像“道路宽度”，表示网络的容量；
- 速率像“车速”，表示实际传输速度。

如果网络带宽很大，但同时有很多设备在占用网络，实际速度也可能变慢。

>[!TIP]
>网络里的“地址”和“名称”往往是分开的：IP 地址是机器识别的数字地址，域名和 SSID 则是人更容易记住的名字。
