## 4 网络

本章节系统介绍从网络基本概念、日常使用，到安全防护与排错修复的完整流程。适合刚开始学习电脑网络的读者，也适合希望掌握家庭网络维护方法的用户。

- [4.1 网络基础知识](网络基础知识.md)：解释什么是局域网、互联网、IP 地址、DNS、路由器等核心概念。
- [4.2 网络使用](网络使用.md)：讲解家庭 Wi-Fi、移动网络、公共 Wi-Fi、浏览器使用、下载上传与常见网络应用。
- [4.3 网络安全](网络安全.md)：介绍如何保护 Wi-Fi、浏览器安全、密码管理、邮件防骗、设备隐私与常见攻击类型。
- [4.4 网络检查和修复](网络检查和修复.md)：提供逐步诊断思路、命令工具、路由器检查与重置网络的方法。

> 本章节强调“先理解再操作”，让你不仅会用网络，还能知道为何出现问题以及如何安全地修复它。
